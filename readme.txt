A Plugin to integrate portfolios

== Description ==
Gravitation portfolios, is a plugin to display portfolios on your site, with a page template, or shortcodes is mainly for bootstrap ready templates.

== Installation ==
Download and install from WordPress.org.
Configure as you see fit, under Gravitation / Gravitation portfolios.
1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place [gravitaiton_portfolios] in your templates
